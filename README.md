## NationIO is a guessing game of countries.

The game will generate a list of 5 countries, each country will have 5 hints available, using the [restcountries API](https://restcountries.com/v3.1).
 The hints will be revealed one by one, and each revealed hint will lower the score for the round.

For example:
```
Guess the country!
Hint 1: <hint goes here>
Hint 2: <hint goes here>
Hint 3: <hint goes here>
Hint 4: <hint goes here>
Hint 5: <hint goes here>

Enter your guess: <guess goes here>
```
